# ⚾ LIDOM League Manager

Sistema de gestión de liga de béisbol inspirado en LIDOM.

## 🚀 Tecnologías

- React + TypeScript
- Node.js + Express
- SQLite

## 📌 Funcionalidades

- Gestión de equipos
- Gestión de jugadores
- Registro de juegos
- Tabla de posiciones automática

## 📂 Estructura

- /frontend → Aplicación React
- /backend → API REST

## 🎯 Objetivo

Proyecto para portafolio como desarrollador full-stack.
